    Features:
  • The chess board changes size as per the browser window’s width (and height).
  • Everything on this page is made by Yashdeep Singh Fauzdar, except for the chess pieces (taken from Lichess).
  • Full FEN support.
  • Detects invalid FENs.
  • Board updates whenever you click out of the FEN input box.
  • System fonts.
  • Dark mode.
  • Ability to flip board.
  • Ability to drag and drop pieces along with clicking.
    Expected soon:
  • Full PGN support.
  • A settings menu.
  • Its own simple chess engine, made by Yashdeep Singh Fauzdar in plain JavaScript.
  • Working evaluation bar.
  • A menu bar, along with web pages on all chess rules.
  • Store settings in browser.
